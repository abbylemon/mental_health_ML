Data

The data used for this project comes from the OSMI Mental Health in Tech survey. This is an annual survey used to conduct research about mental health, focusing on measuring attitudes, perceptions, and prevalence within the tech community.

The survey results from 2014 and 2016-2019 are stored as csv files on kaggle. For this project, csv files were downloaded, converted into dataframes, and loaded into a Postgres database hosted on AWS. For more information on the ETL process, see the ETL page.

Values were saved as null in the database if a survey question was not answered, or if a year had additional questions that other years did not have. Each row includes a year column to distinguish between survey results from different years.

To retrieve the survey results from the database, an API was built using Flask and SQLAlchemy. The following table shows a small subset of the data that is currently in the database.
