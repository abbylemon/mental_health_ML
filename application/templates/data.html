Data

The data used for this project comes from the OSMI Mental Health in Tech survey. This survey is an annual survey used to get information about mental health and how mental health is viewed and perceived by employees within the tech community.

The survey results from 2014-2019 are stored as csv files on kaggle. For this project, we downloaded the csv files, converted them into dataframes, and loaded them into a Postgres database hosted on AWS. For more information on the ETL process, see the ETL page.

If a survey question was not answered or if some years have additional survey questions that other years do not have, then the value is saved as null in the database. Each row includes a year column to distinguish between survey results from different years.

To retrieve the survey results from the database, we built an API using Flask and SQLAlchemy. The following table shows a small subset of the data that is currently in the database.
