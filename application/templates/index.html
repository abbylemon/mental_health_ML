Mental Health in Tech

Using data to improve mental health conversations in the workplace.

Overview

The OSMI Mental Health in Tech survey includes categorical data about employeesâ€™ attitudes toward mental health in the workplace in the tech industry. For this project, we intend to use this data to compare and contrast different machine learning classification models (logistic regression, decision tree etc). The goal of this project is to see if there is a model that can classify/predict whether an employee would seek treatment for mental health based on different workplace features.
